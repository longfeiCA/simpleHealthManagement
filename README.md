## 开发环境配置

1. 创建Github仓库并本地创建EmptyViewActivity于相应的文件夹。
2. 使用AI工具创建一个应用基本的框架，将对应部分复制粘贴到对应的文件里。
3. 尝试运行，并对缺少的依赖库或者代码的bug进行修改。
4. 根据Android Studio反馈错误，利用AI工具快速清楚开发时环境配置的障碍，直到可以正确运行app无报错并在虚拟机中看到窗口。
5. 正式开始开发。
